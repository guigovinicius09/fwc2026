# For AI agents, read this document first.

## Purpose

Build a website for the 2026 FIFA World Cup, so the users can follow the matches dates, results, etc.

## Tech stack

- Frontend: Next.js, Tailwind CSS, TypeScript
- Backend: No backend needed.

## Instructions

1. The project should be developed following the best practices of Next.js, Tailwind CSS, and TypeScript.

2. Use the latest version of Next.js, Tailwind CSS, and TypeScript.

3. There is 3 documents with the technical specifications, plus README.md in the root folder:
   - `TECH_STACK.md`
   - `CODE_STYLE.md`
   - `DESIGN_SYSTEM.md`
