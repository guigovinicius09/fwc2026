# :trophy: Fifa World Cup 2026

## :mag_right: 1. Overview

FIFA World Cup 2026 is an informative web application developed to centralize information about the 2026 FIFA World Cup in a modern and intuitive interface.

The system allows users to view groups, participating teams, match schedules, stadiums, and host cities of the tournament.

The initial focus is on delivering a fast, static frontend experience, hosted on Vercel, with future expansions to interactive geographic views using Globe 3D.

## 2. :bangbang: Problem Statement

Information about the FIFA World Cup 2026 is often distributed across multiple websites, making it difficult for users to quickly access groups, matches, teams, stadiums, and host cities in a single place.

This project aims to centralize this information through a modern and intuitive user experience.

## 3. :busts_in_silhouette: Target audience:

- Football fans
- Followers of the 2026 World Cup
- Users interested in stadiums and host cities
- Students and developers interested in modern web technologies

## 4. :dart: Goals

### 4.1 Main Objective:

Provide a fast, visually appealing, and accessible platform for exploring information about the 2026 FIFA World Cup.

### 4.2 The MVP should include:

- Homepage with groups
- National teams page
- Matches page
- Stadiums page
- Responsive navigation
- Deployment on Vercel

## 5. :no_entry_sign: Out of scope

> This project focuses exclusively on the frontend.

### The following are not part of the current objectives:

- Dedicated backend
- Database
- Authentication system
- User registration
- Private areas
- Comment system
- Custom APIs
- Administrative panel
- Real-time data processing
- Social functionalities
- Static-first architecture
- Accessibility (WCAG basic compliance)

## 6. :gear: Features

### 6.1 Groups

**Description:** Display the 48 national teams organized into official groups.

User can:

- View groups
- View flags
- Identify group composition

### 6.2 Matches

**Description:** Display all matches of the competition.

User can:

- View date
- View time
- View stadium
- View city
- View participating teams

### 6.3 Stadiums

**Description:** Display the stadiums used in the competition.

User can:

- View stadium
- View capacity
- View location
- View host city
